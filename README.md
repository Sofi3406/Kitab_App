# Kitab App

Repository for the Kitab Flutter application.

## About

This project contains a Flutter app for reading PDF lessons while listening to synced lesson audio, with local notes and playback resume support.

Main app location:

- kitab_app/

## Quick Start

```bash
cd kitab_app
flutter pub get
flutter run
```
